MRH_EVENT_UNK
=============
MRH_EVENT_UNK is used to signal an unknown event, indicating a problem during
event creation and / or sending.

Data Buffer
-----------
This event has no data.

Sender
------
Both user applications and all services.

Reciever
--------
Both user applications and all services.

Request Event
-------------
None.

Response Event
--------------
None.

Remarks
-------
None.
