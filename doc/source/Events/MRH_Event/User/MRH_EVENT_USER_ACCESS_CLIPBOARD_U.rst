MRH_EVENT_USER_ACCESS_CLIPBOARD_U
=================================
MRH_EVENT_USER_ACCESS_CLIPBOARD_U is used to request access to the user 
clipboard file.

Data Buffer
-----------
This event has no data.

Sender
------

Reciever
--------
