MRH_EVENT_USER_ACCESS_MUSIC_U
=============================
MRH_EVENT_USER_ACCESS_MUSIC_U is used to request access to the user music 
folder.

Data Buffer
-----------
This event has no data.

Sender
------
User applications.

Reciever
--------
User platform services.

Request Event
-------------
None.

Response Event
--------------
:doc:`MRH_EVENT_USER_ACCESS_MUSIC_S`.

Remarks
-------
None.
