MRH_EVENT_USER_ACCESS_VIDEOS_U
==============================
MRH_EVENT_USER_ACCESS_VIDEOS_U is used to request access to the user videos 
folder.

Data Buffer
-----------
This event has no data.

Sender
------
User applications.

Reciever
--------
User platform services.

-------------

--------------
